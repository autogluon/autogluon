For PyTorch Users
=================

AutoGluon can serve as a framework agnostic hyperparameter optimization toolkit for deep learning, and more generally be used to tune argument-values of any Python code. Prior to this tutorial, first read the :doc:`Tune Custom Models <../course/index>` tutorial to familiarize yourself with AutoGluon's core API.
