For PyTorch Users
=================

AutoGluon is a framework agnostic HPO toolkit for deep learning, which enables
fast prototyping on any customized python code. Prior to this tutorial, we recommend first reading the `Customize AutoGluon <../course/index.html>` tutorial to get familiar with the core API.
