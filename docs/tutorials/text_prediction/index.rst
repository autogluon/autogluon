Text Prediction
===============

AutoGluon Text will be deprecated in AutoGluon v0.6 and will be completely removed in v0.7.
All functionalities in AutoGluon Text have been supported via `AutoGluon Multimodal <https://auto.gluon.ai/stable/tutorials/multimodal/index.html>`_.
You are recommended to switch to AutoGluon Multimodal and may check the relevant tutorials on how to use it for
text prediction problems.
